CSS Position Summary

Simple table for beginners

Position Meaning Simple Explanation
static Default The normal default position of an element.
relative Move from original place The element moves from its original position, but the original space stays reserved.
absolute Position inside parent The element is placed exactly inside its nearest positioned parent.
fixed Stuck to screen The element stays fixed on the screen even when the page scrolls.
sticky Sticks on scroll The element scrolls normally at first, then sticks when it reaches a set position.
Quick tip: relative is often used as the parent for an absolute child.
